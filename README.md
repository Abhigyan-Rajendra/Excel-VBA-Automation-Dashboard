# Excel-VBA-Automation-Dashboard
# 📊 Learning & Development Management Dashboard (Excel VBA)

A complete **Learning & Development (L&D) Management System** built in **Microsoft Excel using VBA (Visual Basic for Applications)**. This project automates trainer and trainee record management through interactive UserForms and provides a dynamic dashboard for monitoring training activities.

## 🚀 Features

- 👨‍🏫 Trainer Management
  - Add, update, search, and delete trainer records
  - Automatic validation of input fields
  - User-friendly VBA UserForms

- 👨‍🎓 Trainee Management
  - Manage trainee information efficiently
  - CRUD (Create, Read, Update, Delete) operations
  - Search and filter functionality

- 📈 Interactive Dashboard
  - Dynamic KPIs
  - Pivot Tables
  - Charts and Visual Reports
  - Training statistics and summaries
  - Automatic dashboard refresh

- 📂 Excel Automation
  - VBA-powered data processing
  - Automated record handling
  - Form validation
  - Error handling
  - Clean and structured worksheet organization

## 🛠 Technologies Used

- Microsoft Excel (.xlsm)
- VBA (Visual Basic for Applications)
- UserForms
- Pivot Tables
- Pivot Charts
- Excel Form Controls

## 📁 Project Structure
